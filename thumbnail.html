How to Get SharePoint Embed Code:

  1. Navigate to your video in SharePoint
  2. Click the "..." (more options) button on the video
  3. Select "Embed" from the dropdown menu
  4. Copy the iframe code that appears
  5. Paste it into the "Video URL/Embed Code" field in the form

Example embed code format:
<iframe src="https://your-sharepoint-site.com/..." width="640" height="360"></iframe>
